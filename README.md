# [ capston : 캡스톤 디자인 프로젝트 ] : 학교 체육시설 예약 시스템  

### 개요
 - 제목 : 캡스톤 프로젝트   
 - 개발기간 : 2023-03 ~ 2023-08 
 - 사용언어 : Java 11    

### 개발환경 (협업 총인원 : 3명)   
 - Back : SpringBoot / jpa / Mysql    
 - Front : React , Android (협업)   
 - Infra : AWS ( 자동화,무중단 배포 )
